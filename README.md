## 项目结构

```
skill-marketplace/
├── .codebuddy-skill/
│   └── marketplace.json          # 技能市场清单（核心）
├── skills/                       # 技能库目录（100+ 技能）
│   ├── skill-name/
│   │   ├── SKILL.md             # 技能定义文件（必需）
│   │   ├── references/          # 参考文档（可选）
│   │   └── scripts/             # 辅助脚本（可选）
│   └── ...
├── icons/                        # 技能图标
├── skills-standard.md            # 技能开发标准
└── README.md                     # 本文件
```

## 贡献新技能

### 快速流程

1. 创建技能目录：`mkdir skills/my-skill-name`
2. 编写 `SKILL.md`（含 YAML frontmatter）
3. 更新 `.codebuddy-skill/marketplace.json`（按字母顺序插入）
4. 验证 JSON：`jq . .codebuddy-skill/marketplace.json`
5. 提交并推送

### 命名规范

- 目录名使用 **kebab-case**（小写 + 连字符）
- 示例：`apple-notes`、`tencent-docs`、`github-ai-trends`

### 版本号

使用语义版本 `MAJOR.MINOR.PATCH`，仅允许纯数字。

详细规范请参考 `skills-standard.md`。
