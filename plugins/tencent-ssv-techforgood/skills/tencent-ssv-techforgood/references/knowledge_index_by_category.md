## 四、使用规则

1. 本文件只保留在线索引表、申领资格与账号管理三类高频官方资料
2. 用户只问“怎么申领 / 资格是什么 / 账号怎么处理 / 去哪里看官方说明”时，优先读本文件
3. 用户只问产品申领地址时，直接读 `references/digital_toolbox.md`
4. 用户要教程、案例、实施参考、产品帮助文档时，回到 `references/knowledge_index.md`，优先通过“官网 Agent 训练知识库总表”实时匹配并读取命中的在线文档
5. 不在本文件长期固化案例、教程、公众号文章或帮助文档清单
