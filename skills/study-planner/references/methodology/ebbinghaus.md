# 艾宾浩斯遗忘曲线在学习中的应用

## 核心原理

德国心理学家艾宾浩斯研究发现，遗忘在学习后立即开始，且速度先快后慢：

| 时间 | 记忆保留率 |
|-----|----------|
| 20 分钟 | 58% |
| 1 小时 | 44% |
| 8 小时 | 36% |
| 1 天 | 33% |
| 2 天 | 28% |
| 6 天 | 25% |
| 31 天 | 21% |

## 在学习计划中的应用

### 任务间隔复现设计原则

1. **首次学习**：新知识当天接触
2. **短期复现**：次日内复习（对抗快速遗忘）
3. **中期巩固**：3-5 天后再次复习（加固长时记忆）
4. **长期保持**：14 天、30 天、60 天间隔复现

### 模板示例：雅思听力复习节奏

```
Day 1：学习 Section 1 定位词技巧（首次接触）
Day 2：练习剑 14 Test 1（短期复现）
Day 5：复盘错题 + 同义替换总结（中期巩固）
Day 14：重做本周所有错题（长期保持入口）
Day 30：全真模考（综合检验）
```

### 适用场景

- **语言学习**：单词/短语/语法规则
- **考试备考**：公式/概念/解题方法
- **技能习得**：操作步骤/设计模式/代码规范

## 生成计划时的应用规则

当用户的学习计划包含需要记忆的内容时：
1. 在 `daily_tasks` 中插入 `methodology_tip: "艾宾浩斯复现点"`
2. 标注哪些任务是"新学"，哪些是"复习"
3. 确保复习任务出现在关键时间节点（次日/3天后/7天后/14天后）

## AI 注入指南

### 触发条件（满足任一即应用）

- 计划目标含「记忆」「背诵」「单词」「公式」「概念」「术语」
- 计划类型属于：语言考试 / 应试备考 / 资格证书 / 医学法律
- 用户主动提到"容易忘"「记不住」「复习」

### 强度档位

| 档位 | 触发场景 | 复现密度 |
|-----|---------|---------|
| 轻量 | 技能学习类（如编程） | 仅在阶段末安排 review 周 |
| **标准** | 大多数语言/应试场景 | 1/3/7/14/30 天五点复现 |
| 重度 | 医学/法律/历史等大量记忆 | 增加 1/2/4/8 天密集复现 |

### plan.json 注入片段

```json
{
  "methodologies": ["ebbinghaus"],
  "daily_tasks": [
    {
      "task_id": "t-d01-03",
      "title": "新学：雅思高频词 List 5（80 词）",
      "category": "memorize",
      "tags": ["new"],
      "methodology_tip": "ebbinghaus:new"
    },
    {
      "task_id": "t-d02-02",
      "title": "复习：List 5 错词（重点 30 词）",
      "category": "memorize",
      "tags": ["review", "+1d"],
      "methodology_tip": "ebbinghaus:review-1d",
      "depends_on": ["t-d01-03"]
    }
  ]
}
```

### 与其他方法论的协同

- **+ pareto**：先用二八筛出高频考点，再用艾宾浩斯安排复现 → 优先记忆 ROI 最高
- **+ feynman**：每个复现节点搭配一次"用自己话讲"的输出任务，记忆更牢
- **× pomodoro**：单纯番茄不影响艾宾浩斯节奏，可叠加
