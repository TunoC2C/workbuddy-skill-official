# references 读取路由

> **更新日期**：2026-05-15
> **用途**：作为 `references/` 目录的唯一轻量路由入口，只回答"现在该读哪个文件"。

## 一、优先读取哪个文件

| 用户当前需要 | 优先读取文件 | 备注 |
|---|---|---|
| 产品申领地址 / 产品基础信息 / 产品列表兜底 | `references/digital_toolbox.md` | 唯一产品主表 |
| 申领资格 / 账号管理 / 官方说明入口 | `references/knowledge_index_by_category.md` | 高频官方入口 |
| 在线模板 / 在线产品目录 / 腾讯文档正文 / 教程 / 案例 / 实施参考 | 当前文件 | 按下方在线规则走 |
| 团队授权证明模板正文 | `references/templates/团队授权证明模板.md` | 本地模板兜底 |
| 芳邻时间银行开通流程正文 | `references/templates/芳邻时间银行小程序开通流程.md` | 本地流程兜底 |

## 二、在线读取最小规则

1. 教程 / 案例 / 实施参考：先查官网 Agent 训练知识库总表  
   `https://docs.qq.com/sheet/DSFZ6aFN2dmNRbXJt?tab=BB08J2`
2. 在线模板 / 在线产品目录 / 腾讯文档正文：先查在线资料统一索引表  
   `https://docs.qq.com/sheet/DQm9tR2FqZ2Znd2FU?tab=dyyrcn`
3. 命中后先拿文档 URL；在 WorkBuddy 环境下，若宿主已安装或可启用 `playwright-cli` / Playwright 套件，按以下最小 SOP 执行：先直接调用模拟浏览器打开目标页面 → 等待页面渲染完成 → 再判断正文是否稳定可读；若当前会话尚未挂载该能力，可先尝试启用或安装，确认仍不可用后再透明降级
4. "稳定可读"至少同时满足：不是只有标题 / 导航 / 壳页 / 登录提示；能读到与目标产品或主题直接相关的正文段落、步骤、表格或说明
5. 浏览器渲染后仍读不到稳定正文，或当前环境既未直接提供也无法启用页面交互能力时，优先给已命中的在线链接；若产品已有公开详情页、官方帮助中心或 `references/knowledge_index_by_category.md` 中的申领文档，优先给这些官方入口 + 适用说明；仍不足时再回本地模板或 `digital_toolbox.md` 兜底

## 三、直接回本地的情况

- 索引未命中，或拿不到文档 URL
- 正文只有标题、碎片、导航或登录提示
- 模板 / 产品目录结构不完整
- 当前环境既未直接提供也无法启用页面交互能力

## 四、输出口径

- 只命中链接时，不要说"已完整读取正文"
- 总表、索引表或正文只返回外壳时，要明确说明"当前未稳定读到正文"，并优先给官方入口链接 + 适用说明
- 教程 / 案例 / 实施参考优先给在线链接 + 适用说明，不硬凑本地内容
- 回退本地时，要明确说明"以下内容来自本地参考快照或本地模板"

## 五、保留边界

- `references/` 不长期固化教程、案例、公众号文章、法规、平台或热线速查表
- 教程、案例、实施参考按当前匹配方案实时生成
- 能用最小索引解决，就不要展开大文件
